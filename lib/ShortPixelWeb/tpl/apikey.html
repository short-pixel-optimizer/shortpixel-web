Quick guide

Step 1

If you don't have an API Key, sign up here. It's free and it only takes one minute.

Step 2

Fill in the API Key in the field above and press Validate. The API Key will be saved on your site in the shortpixel-php/shortpixel.ini file.

Step 3

After you input your API Key, the folder selection page will be available. Please select the folder that you want to optimize and press the Optimize Now button. Please make a backup of your images folder and make sure that the folder you optimize is writable.
